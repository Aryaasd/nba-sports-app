# NBA Sports App

A Streamlit app to view and compare NBA player statistics using the `nba_api`.

## Features

- Select an NBA player and view their statistics  
- Compare players side-by-side  
- Interactive visuals built with Altair and Pandas  
- Built with Streamlit for easy web deployment  

## Installation

```bash
pip install -r requirements.txt
streamlit run sports_app.py
